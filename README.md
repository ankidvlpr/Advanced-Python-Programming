# Advanced Python Programming

A hands-on repository for learning advanced Python concepts through structured practice, object-oriented thinking, and project-based problem solving.

## Overview
This repository is the second part of a two-repo learning journey:

- Introduction to Python Programming - fundamentals, core syntax, and problem-solving practice
- Advanced Python Programming - object-oriented programming, stateful design, abstraction, and larger problem-solving exercises

The goal of this repo is not only to write working code, but to build a deeper understanding of how Python programs are designed, organized, and extended as they become more complex.

This repository focuses on:

- object-oriented thinking
- class design and state management
- reading and tracing class-based code
- understanding shared state vs instance state
- building stronger problem-solving depth through practice

## Repository Map
- `OOP.py` - consolidated OOP practice and class-based examples
- `oop_foundations` - core class exercises such as counters, email parsing, text statistics, and user sessions
- `stateful_oop_patterns` - practice with configuration overrides, feature flags, caching, and state-oriented class behavior

## Purpose
This repository grows as I move from Python fundamentals into more structured and advanced programming topics. It serves as both a practice archive and a record of steady progress while I strengthen my understanding of object-oriented programming and real-world Python design patterns.
